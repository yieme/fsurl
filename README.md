# fsurl

Fetch JSON data from File System or URL

<!-- [![build status](https://secure.travis-ci.org/yieme/fsurl.png)](http://travis-ci.org/yieme/fsurl) -->

## Installation

This module is installed via npm:

```sh
npm i fsurl -g
```

## Example Usage

```js
fsurl --help
```
